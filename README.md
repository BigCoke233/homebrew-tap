# BigCoke233 Tap

## How do I install these formulae?

`brew install bigcoke233/tap/<formula>`

Or `brew tap bigcoke233/tap` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "bigcoke233/tap"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
